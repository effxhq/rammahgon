## Codes

| Code | Meaning                                                                                                                                   |
| ---- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| 400  | Request is invalid. Take another look at your request and make sure it matches our API.                                                   |
| 403  | An authentication error. Either you forgot to attach an API Key or it is invalid.                                                         |
| 500  | An Internal Server Error. Something went wrong on our end. Contact <a href="mailto:support@effx.com">support</a> if the problem persists. |
